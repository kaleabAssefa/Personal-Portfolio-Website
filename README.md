# Personal-Portfolio-Website

This is my personal portfolio website built using **HTML and CSS**. It showcases my skills, education, projects, and contact information as a Full-Stack Developer Software Engineer.

## Features

- Responsive multi-page website
- Navigation between pages (About, Education, Projects, Skills, Contact)
- Projects section with images and descriptions
- Education section with structured table
- Contact form with dropdowns, radio buttons, and checkboxes

## Technologies Used

- HTML5
- CSS3
- Flexbox & Grid
- Responsive Design
